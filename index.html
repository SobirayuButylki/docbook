Docbook

Short-Attention-Span Docbook is a docbook build chain for OSX, Linux, and Windows. Contains everything you need to get started as an author as long as you have Java and Ruby installed.

Want to write a book?

This toolchain makes it simple to get started publishing with the open-source DocBook format. It's a collection of scripts that generate and build book projects. All you have to do is supply the content in DocBook format, and this toolchain will help you build HTML, txt, PDF, and epub formats for you.
